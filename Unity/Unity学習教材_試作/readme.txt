Unityをはじめてつかう人を対象に，Unityの使い方を学ぶための問題を試作してみました．
教える項目はtransform.translate関数にしました．

左側に問題文とソースコードと解答欄があり，ソースコードは右側の画面のPlayer（青色の立方体）にアタッチされているものと仮定しています．
ソースコードの赤字になっている部分に当てはまると思う数値を解答欄に入力して実行ボタンを押すと，
その数値が代入されたソースコード通りにPlayer（青色の立方体）が動きます．

正しい数値が入力されていればPlayerがGoal（オレンジ色の球体）のところにたどり着き正解となります，
間違っているとどの部分が間違っているかを教えてくれます．

右側の画面のPlayerやGoalをクリックするとその座標が示されます．
またマウスホイールでカメラの距離，右クリックボタンを押したままドラッグするとカメラの向きを変えられます．

リセットボタンで問題をリセットして最初からにします．
説明ボタンでtransform.translate関数について説明しているページに遷移します．

実行ファイルはbuild → build.exeです．
ソースコードはAsset → Scriptです．
